# Pokedex-Guide Clone

Basit bir Pokedex frontend klonu.

İçerik:
- `index.html`, `app.js`, `style.css`

Yerel git deposu oluşturuldu. GitHub'a itme (push) için iki seçenek:

1. GitHub CLI kullanarak otomatik oluşturma ve push (önerilen):
```
gh repo create <kullanici>/<repo-adi> --public --source=. --push
```

2. Manuel: GitHub'da repo oluşturup aşağıdaki komutları çalıştırın:
```
git remote add origin https://github.com/<kullanici>/<repo-adi>.git
git branch -M main
git push -u origin main
```
